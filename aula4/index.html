Execução

comando: 

 php -S localhost:8080 src/router.php
saída: 
 PHP 7.4.11 Development Server (http://localhost:8080) started

Rotas Pessoa

  • /pessoa - [GET]: Retorna uma lista de pessoas.
    Requisição (json object), opcional: 
     {"id":"valor","nome":"valor","email":"valor"}
    obs.: Os valores informados no json da requisição serão usados como filtro. Para ignorar o filtro defina os valores como null.
    Resposta (json object): 
     {"data":[{"id":"valor","nome":"valor","email":"valor","senha":"valor"}}],"status":valor}

  • /pessoa/salvar - [POST]: Salva pessoa.
    Requisição (json object): 
     {"nome":"valor","email":"valor","senha":"valor", "confSenha":"valor"}
    Resposta (json object): 
     {"pessoa":{"nome":"valor","email":valor,"senha":valor},"status":valor}

  • /pessoa/visualizar - [GET]: Retorna página de visualizar pessoas.

  • /pessoa/criar - [GET]: Retorna página do formulário de cadastro de pessoa.
